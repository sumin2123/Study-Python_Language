## 문자열 연산하기

### 1. 더해서 연결하기(Concatenation)
```python
head = "python"
tail = " is fun!"
print(head + tail)

=> python is fun!
```
### 2. 곱하기
```python
a = "python"
b = a * 2
print(b)

=> pythonpython
```

### 3. 길이 구하기
len() 함수를 사용하면 문자열 길이를 구할 수 있다.
```python
a = "I want go home"
print (len(a))

=> 14
```
