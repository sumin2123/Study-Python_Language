## 문자열 연산하기

### 1. 더해서 연결하기(Concatenation)
```python
head = "python"
tail = " is fun!"
print(head + tail)

=> python is fun!
```
### 2. 곱하기
```python
a = "python"
b = a * 2
print(b)

=> pythonpython
```

### 3. 길이 구하기
len() 함수를 사용하면 문자열 길이를 구할 수 있다.
```python
a = "I want go home"
print (len(a))

=> 14
```

## 인덱싱과 슬라이싱
* 인덱싱 ? "가리킨다" 라는 의미
* 슬라이싱 ? "잘라낸다" 라는 의미
### 인덱싱
1. "파이썬은 0부터 숫자를 센다."
2. a에 저장한 문자열의 각 문자마다 번호를 매겨 보면 다음과 같다.
3. a[번호]는 문자열 안의 특정한 값을 뽑아내는 역할
```python
a = "Life is too short, You need Python"

a[0]
=>'L'

a[12]
=>'s'

a[-1]
=>'n'

```
