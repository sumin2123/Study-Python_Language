#### [ 변수 이름 짓기 ]
  < 필수 약속 >
 1. 문자와 숫자, __ 를 사용할 수 있다.
 2. 대문자와 소문자를 구분한다.
 3. 공백은 사용할 수 없다.
 4. 숫자로 시작할 수 없다.
 5. 미리 예약된 이름은 사용할 수 없다.
