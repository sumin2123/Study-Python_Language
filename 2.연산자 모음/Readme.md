# 기본 연산자 
--------------------------------------------------------
## 1. 산술 연산자 (Arithmetic Operators):
* a = 10, b = 20, c =3 이라 가정한다.

- 더하기 ( + )   a + b = 30  \
- 빼기   ( - )   a - b = -10  \
- 곱하기 ( * )   a * b = 200  \
- 나누기 ( / )   b / a = 2.0  \
- 나머지 ( % )   b % a = 0  \
- 제곱  ( ** )   a ** c = 1000  \
- 몫    ( // )   a // c = 3
